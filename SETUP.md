# Setup Instructions

## 1. Supabase Setup
1. Go to https://zqjsgdbasugsicabhvof.supabase.co
2. SQL Editor → Run `supabase/schema.sql`
3. Storage → Create bucket: `user-images` (Public)
4. Authentication → Enable Email auth

## 2. Environment
Check `.env.local` has all keys

## 3. Run
```bash
npm install
npm run dev
```

## 4. First User
Visit http://localhost:3003/signup

## Done!

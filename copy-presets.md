# Preset Images Setup Guide

## Manken Resimleri (Models)
Aşağıdaki resimleri `public/presets/models/` klasörüne kopyalayın:

1. **model1.jpg** - Pembe elbiseli kadın (ilk resim)
2. **model2.jpg** - Kırmızı elbiseli sarışın kadın
3. **model3.jpg** - Siyah mayolu, dövmeli kadın
4. **model4.jpg** - Siyah ceketli kadın

## Kıyafet Resimleri (Clothes)
Aşağıdaki resimleri `public/presets/clothes/` klasörüne kopyalayın:

1. **dress1.jpg** - Siyah triko elbise (uzun kollu)
2. **pants1.jpg** - Bej/Ten rengi pantolon
3. **pants2.jpg** - Gri pantolon
4. **jacket1.jpg** - Yeşil/Haki cepli ceket
5. **jacket2.jpg** - Siyah ceket
6. **jacket3.jpg** - Kahverengi/Vizon kadife ceket
7. **dress2.jpg** - Siyah kemerli kolsuz elbise
8. **dress3.jpg** - Bordo çiçek desenli elbise

## Manuel Kopyalama
Resimleri Windows Explorer'da bu klasörlere sürükleyip bırakabilirsiniz:
- `c:\Users\seyha\CascadeProjects\comfyuiweb\public\presets\models\`
- `c:\Users\seyha\CascadeProjects\comfyuiweb\public\presets\clothes\`

## Veya Komut Satırı ile:
```bash
# Model resimleri
cp "/path/to/your/model1.jpg" public/presets/models/model1.jpg
cp "/path/to/your/model2.jpg" public/presets/models/model2.jpg
cp "/path/to/your/model3.jpg" public/presets/models/model3.jpg
cp "/path/to/your/model4.jpg" public/presets/models/model4.jpg

# Kıyafet resimleri
cp "/path/to/your/dress1.jpg" public/presets/clothes/dress1.jpg
cp "/path/to/your/pants1.jpg" public/presets/clothes/pants1.jpg
cp "/path/to/your/pants2.jpg" public/presets/clothes/pants2.jpg
cp "/path/to/your/jacket1.jpg" public/presets/clothes/jacket1.jpg
cp "/path/to/your/jacket2.jpg" public/presets/clothes/jacket2.jpg
cp "/path/to/your/jacket3.jpg" public/presets/clothes/jacket3.jpg
cp "/path/to/your/dress2.jpg" public/presets/clothes/dress2.jpg
cp "/path/to/your/dress3.jpg" public/presets/clothes/dress3.jpg
```
